Plume

A particle-flow field in one Kāra source — the per-frame compute fans out across every CPU core, in a browser tab, with no async and no threading code. Move the mouse over the canvas to steer the flow; click to pin a glowing swirl where you leave it.

The pointer and clicks arrive as typed channels — Channel[PointerEvent] and Channel[ClickEvent] — each drained every frame with a plain try_recv; no addEventListener callback in the Kāra source. The render loop is a blocking loop { frames.recv(); … }; each frame's rows split across a Web Worker pool by TaskGroup.spawn.
