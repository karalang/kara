Iris

An image editor in one Kāra source — the filter kernels compile unchanged to a native binary and to browser WASM, and every frame's pixels are computed across a Web Worker pool with no async and no threading code. Press 1–6 (or click) to switch filter.

Where is the threading code? There isn't any — the compiler found it. The render loop is a plain blocking loop { frames.recv(); /* drain keydown */ render_frame(filter); }; each frame's output rows are split across a Web Worker pool by TaskGroup.spawn, and the filter math lives in filters.kara, shared byte-for-byte with the native build.
